# Русский ЕГЭ Игра

Это интерактивная веб-игра для подготовки к ЕГЭ по русскому языку в стиле Duolingo. Игра включает в себя различные темы, такие как орфография, пунктуация, лексика и грамматика.

## Особенности

- Интерактивные вопросы по ключевым темам ЕГЭ по русскому языку
- Система прогресса с подсчетом XP и серий
- Объяснения к каждому ответу
- Адаптивный дизайн для разных устройств
- Темы: орфография, пунктуация, лексика, грамматика

## Структура проекта

- `index.html` - основная HTML-структура игры
- `styles.css` - стили для игры
- `script.js` - логика игры и вопросы
- `server.js` - сервер для запуска игры локально

## Установка и запуск

1. Убедитесь, что у вас установлен Node.js
2. Установите зависимости: `npm install`
3. Запустите сервер: `node server.js`
4. Откройте в браузере: `http://localhost:3000`

## Использование

1. Выберите тему из предложенных
2. Отвечайте на вопросы, нажимая на правильные варианты
3. Получайте объяснения к каждому вопросу
4. Следите за прогрессом и XP
5. Повторите для другой темы

## Технологии

- HTML5
- CSS3
- JavaScript (ES6)
- Express.js (для сервера)
